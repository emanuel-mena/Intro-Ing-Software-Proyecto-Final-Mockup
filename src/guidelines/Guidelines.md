Use the following visual design system for all screens you generate:

COLOR PALETTE:
- Primary: #1B4FA8 (Government Blue)
- Secondary Blue: #3D79F5
- Accent Blue: #2D9CDB
- Success: #2BA84A
- Warning: #F2C94C
- Error: #D64545
- Neutrals: #1A1A1A, #4A4A4A, #8F8F8F, #CFCFCF, #F5F5F5, #FFFFFF

TYPOGRAPHY:
- Use “Inter” as the main font.
- Headings: 24–28 px
- Subheadings: 20–22 px
- Body text: 14–16 px
- Buttons and labels: 14–15 px, semibold

SPACING SYSTEM:
- Use spacing steps: 4, 8, 12, 16, 24, 32 px
- Card radius: 8 px
- Button height: 48 px
- Input height: 44–48 px
- Icon size: 20–24 px

COMPONENTS:
- Primary button (solid blue)
- Secondary button (light gray)
- Ghost button (text-only)
- Input fields, dropdowns, textareas
- Card layout for lists
- Status badges: Pendiente (gray), En Proceso (blue), Alta (red), Resuelto (green)
- Mobile bottom navigation bar for citizen role

INTERACTION PATTERNS:
- Citizen views: simple navigation, card-based UI
- Staff views: dense lists with filters
- Admin views: dashboard layouts with charts and metrics

All UI text MUST be in Spanish.

Generate consistent, clean wireframes that follow this design system.
